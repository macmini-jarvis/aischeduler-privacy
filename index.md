## 1. 개인정보 수집 및 이용

### 1.1 수집하는 개인정보
AI 스케줄러는 다음 정보를 수집합니다:

- **일정 정보:** 사용자가 입력한 일정 제목, 날짜, 시간, 장소
- **위치 정보:** 위치 기반 알림 기능을 위한 현재 위치 (사용자 동의 시)

### 1.2 수집 방법
- 사용자가 앱 내 AI 채팅 기능에 직접 입력
- iOS 위치 서비스를 통한 위치 정보 수집 (권한 승인 시)

### 1.3 수집 목적
- **일정 정보:** AI 기반 일정 생성 및 스마트 알림 제공
- **위치 정보:** 목적지까지의 거리 계산 및 도착 가능 여부 알림

### 1.4 보관 기간
- **일정 정보:** 사용자 기기 내에만 저장 (iCloud 미동기화)
- **Firebase 전송 데이터:** 처리 즉시 삭제 (서버 저장 없음)
- **위치 정보:** 일시적 처리 후 즉시 삭제 (서버 저장 없음)

## 2. 제3자 제공

### 2.1 Firebase (Google)
AI 스케줄러는 일정 생성 및 사용자 인증을 위해 **Firebase**를 사용합니다.

**사용 서비스:**
- Firebase Authentication (Apple 로그인)
- Firebase Cloud Functions (AI 일정 파싱)

**전송 정보:**
- 사용자가 입력한 텍스트 (예: "내일 오후 3시에 회의")
- 인증 토큰 (로그인 시)

**전송 목적:**
- 자연어를 구조화된 일정 데이터로 변환
- 사용자 인증 및 구독 관리

**Google 개인정보 처리방침:**
[https://policies.google.com/privacy](https://policies.google.com/privacy)

**데이터 보관:**
- Cloud Functions는 요청 처리 후 데이터를 보관하지 않음
- 인증 정보는 Firebase Auth에 보관 (계정 삭제 시 삭제)

**거부 방법:**
- 게스트 모드로 로그인 없이 사용 가능
- AI 채팅 대신 수동 일정 추가 기능 사용

## 3. 개인정보 보호

### 3.1 보안 조치
- Firebase 인증은 Apple Sign-In 기반 보안 처리
- HTTPS 통신으로 데이터 전송
- 사용자 기기 외부에 일정 데이터 저장 안 함

### 3.2 데이터 삭제
- 앱 삭제 시 모든 일정 데이터 자동 삭제
- 백업 없음 (iCloud 동기화 미지원)
